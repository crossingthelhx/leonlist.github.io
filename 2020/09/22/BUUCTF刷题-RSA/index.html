[NPUCTF2020]共 模 攻 击

题目

分析

先解密 hint 文件。

先通过共模攻击求出 c。

对于 c = pow(m, 256, p) ，可以通过 SageMath 的取模求根来解。

求解得到：m.bit_length() < 400

c1 p = m % n
c2 p = m % n
从而可以推出:
c1 = m % p
c2 = m % q
接着
k1p + c1 = m
k2q + c2 = m
两式相乘得:
m2 = k1 k2 p q + c1 k2 q + c2 k1 p + c1 c2
由 1 c2 2 c1 得:
k1 c2 p + c1 c2 = c2 m
k2 c1 q+c1 c2 = c1 m
由 3-4-5得:
m2 -m(c1+c2)+c1 c2 = k1 k2 p q
即:
m2 - m(c1 +c 2) + c1 * c2 = 0 (mod n)

然后用 Sage 解方程。
